## 📌 What MCP Means

MCP stands for Model Context Protocol. It is a standard way for AI apps to connect to tools and data.

In plain terms, MCP helps an AI app:

- Reach outside services
- Use local or remote tools
- Pull in data from other places
- Do more than answer text prompts

## 🔎 Common Uses
