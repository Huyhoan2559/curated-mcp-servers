# Contributing to Curated MCP Servers

Thanks for your interest in contributing! We welcome all submissions of MCP servers, clients, and frameworks.

## How to Add an MCP Server

1. **Fork** this repository
2. **Edit** `README.md` and add your server in the appropriate section
3. **Submit** a pull request

## Formatting Guidelines

Add your entry in **alphabetical order** within the correct section:

- **Official Servers** — Production-ready servers maintained by the company/org that built them
- **Community Servers** — Servers built by the community
- **Clients** — MCP client implementations
- **Frameworks** — Libraries and frameworks for building MCP servers

### Entry Format

```markdown
- **[Server Name](https://github.com/your/repo)** - A brief description of what it does. [Website](https://your-website.com)
```

### Rules

- Description should be **one sentence**, clear and concise
- Link to a **public repository** (GitHub, GitLab, etc.)
- The server must be **functional** — no empty repos or placeholders
- No duplicate entries — search the list first
- Keep entries in **alphabetical order** within their section

## Want to Be Featured?

If you'd like your MCP server to be highlighted in the Featured section, open an issue with the `featured-request` label explaining why your server stands out.

## Reporting Issues

Found a broken link or outdated entry? Open an issue or submit a PR to fix it.

## Code of Conduct

Be respectful and constructive. We're all here to build a useful resource for the MCP community.
